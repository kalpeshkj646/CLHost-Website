Install CLi

Requirements

  • Windows 11, MacOS, Linus
  • Node.js -> V12 or higher

Install

Mac (Homebrew)

$ brew install CLHost-cli

NPM

$ npm install CLHost-cli

Yarn

$ yarn install CLHost-cli
